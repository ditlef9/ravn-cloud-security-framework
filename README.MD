🐦‍⬛ Ravn Cloud Security Framework 🐦‍⬛

A production-ready cloud security framework for building secure environments in 
- **Microsoft Azure**, 
- **Amazon Web Services (AWS)**, and
- **Google Cloud Platform (GCP)**.

The framework combines industry best practices, Infrastructure as Code templates, security checklists, and deployment guides to help organizations build secure cloud foundations.

## 1. 🚀 Quick Start

### Azure (Bicep)

For example to deploy to Azure with Bicep then do the following:

```bash
git clone https://github.com/<user>/cloud-security-framework.git

cd azure/bicep

az deployment sub create \
    --location westeurope \
    --template-file main.bicep
```

## 3. 🛡️ Design Principles

The framework is based on the following principles:

- Secure by default
- Least privilege
- Zero Trust
- Infrastructure as Code
- Immutable infrastructure
- Defense in depth
- Continuous compliance
- Automation over manual configuration
